## Installation

### composer
<p>Execute this command after clone repository to install dependencies</p>

```
	composer install
```

### Migrations
<p>After created database, use this command to create tables</p>

```
	php artisan migrate
```

## Test

<p>To test the application before using it, use this command</p>

```
	php artisan test
```
